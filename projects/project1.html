Project Overview

This project is a personal portfolio website I designed and developed to showcase my skills, experience, and projects. It’s fully responsive and features interactive animations and clean layout.

🔧 Technologies Used

  • HTML5 for structure
  • CSS3 for design and animations
  • JavaScript for interactivity
  • GitHub Pages for deployment

✨ Features

  • Mobile-first responsive design
  • Smooth scroll and animated elements
  • Contact form with input validation
  • Project showcase cards with detail pages
  • Clean navigation and sticky header

🚀 Future Improvements

  • Add a backend to handle the contact form (e.g., using Node.js)
  • Include a CMS or JSON-based project loader
  • Multilingual support
